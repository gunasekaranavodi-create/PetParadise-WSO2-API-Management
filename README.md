# PetParadise WSO2 API Management

## Project Overview
Pet Paradise Animal Clinic API Management Platform built using WSO2 API Manager 4.7.0.

## WSO2 Product Used
- **WSO2 API Manager 4.7.0**

## Technologies
- WSO2 API Manager 4.7.0
- Spring Boot 4.0.2
- MySQL 8.0
- Java 21

## API Endpoints Exposed
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/pets | Get all pets |
| GET | /api/appointments | Get all appointments |

## What Was Demonstrated
- API Creation via Publisher Portal
- API Publishing to Developer Portal
- Application Subscription
- OAuth2 Key Generation
- API Gateway Testing (200 OK response)


## Setup
1. Start WSO2 API Manager
2. Run Spring Boot backend on port 8080
3. Access Publisher Portal: https://localhost:9443/publisher
